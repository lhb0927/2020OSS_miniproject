#### Program functions

**카페 메뉴 관리 프로그램**
==========================
주제: CRUD 프로그램의 일종으로 카페 메뉴를 관리를 새롭게 등록하고, 조회하고, 수정하고, 삭제하는 프로그램이다.

## 구조체
* 메뉴 이름

* 가격
 
 * 사이즈(S/M/L)
 
 * ICE/HOT

## 함수기능

  1. 조회
  2. 추가
  3. 수정
  4. 삭제
  5. 저장
  6. 검색(가격, 크기, 타입)
